# 火车出行记录管理系统 - 更新日志

## 版本 11.2 (2026-03-24) - 数据筛选与多选统计增强

### 🎯 重大更新
- **表格首行插入** - 修改了数据录入逻辑，新增行默认插入在表格最上方，契合“最新优先”的展示逻辑
- **多维度检索系统** - 在管理页面引入高级筛选条，支持年份、车次、型号、站点、城市等跨字段模糊搜索
- **勾选统计模式** - 表格新增多选框（支持一键全选），勾选后实时汇总显示选中项的里程、费用与时长

### ✨ 新增功能
- **统计悬浮框** - 选中行时动态弹出统计条，清空选择或退出模式后自动隐藏
- **配置模板文件** - 提供了 `js/config.example.js` 示例配置，明确了 Git 忽略与本地私密配置的最佳实践

### 🔧 改进与修复
- **AI 问答请求修复** - 修正了 `config.js` 中三方 AI 接口地址引用的变量名错误，恢复问答功能
- **搜索防冲突优化** - 修复了全选复选框点击事件冒泡导致触发列表排序的问题
- **清理测试文件** - 移除了项目中残留的 `test_ai.js` 等临时调试脚本

## 版本 11.1 (2026-03-04) - 回放性能优化与数据同步修复

### 🎯 重大更新
- **动画回放性能极致优化** - 重写核心绘制引擎，从 O(N²) 降至 O(1) 复杂度，长路线回放丝般顺滑
- **火车/飞机切换数据修复** - 彻底解决模式切换时统计、图表、地图数据不同步的问题

### ✨ 新增功能
- **动态播放速度** - 回放速度根据路线长短自动调节：短途"嗖"一下画完，长途视觉耗时更真实
- **智能防抖跟随** - 「画面跟随移动」增加 300ms 节流，运镜更平滑不抖动

### 🔧 改进与修复
- **修复模式切换数据丢失** - 切换火车/飞机时强制清空旧表格行，避免数据残留和混合显示
- **修复初始化顺序错误** - 确保先计算年份范围再渲染统计面板和图表，避免空白展示
- **修复跨页导航统计丢失** - 引入 `safeInitStatsAndCharts` 智能等待机制，解决 Chart.js 竞态条件
- **修复 UI 控件层级问题** - 降低线宽控制和年份图例的 z-index，防止在动画回放弹窗中穿透遮罩
- **Leaflet 画线优化** - 使用原生 `addLatLng` 替代全量 `setLatLngs`，增量绘制效率提升数十倍

## 版本 11 (2026-03-01) - 交互体验与 UI 增强

### 🎯 重大更新
- **全局 ESC 键响应** - 统一了全站所有弹窗、遮罩层和右键菜单的 ESC 键盘关闭逻辑，大幅提升交互一致性
- **地图交互增强** - 为地图路径（Polyline）新增右键菜单，提供"查看详情"和"重新画线"功能
- **行程详情弹窗** - 新增精美的行程详情 Modal，自动计算均速并格式化显示时长（小时+分钟）

### ✨ 新增功能
- **路径右键菜单** - 支持右键点击地图线路直接操作
- **全局统一关闭** - 支持 ESC 关闭：AI 对话、年度报告、回放、自定义路线、地址管理、云端设置等 10+ 个 UI 元素

### 🔧 改进与修复
- **修复重绘阴影 Bug** - 彻底解决了重新绘制线路时出现"双重线条"的重影问题
- **UI 比例优化** - 主页侧边栏宽度增加 30px，提升内容展示空间
- **默认视野调整** - 主页地图默认缩放级别调整为 3，提供更宏观的全球/全国视野
- **数据绑定修复** - 修正了行程详情中车次、票价等字段显示不正确的 Bug
- **动画清理优化** - 确保在各种关闭方式下（ESC、按钮或背景点击）都能正确清理回放定时器和地图临时对象

## 版本 10 (2025-12-16) - JavaScript 模块化重构

### 🎯 重大更新
- **JavaScript 架构重构** - 从单文件重构为专业的模块化架构
- **代码组织优化** - 创建 pages/modules/services/utils 四层结构
- **工具函数提取** - 独立工具模块，提高代码复用性
- **统计模块拆分** - 统计功能独立为单独模块

- 文件结构更清晰，职责分离明确
